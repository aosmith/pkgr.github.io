Go

Detection

A Go application will be detected if it contains a .go file.

Requirements

Your application should have either:

  • a .godir file containing the name of your application. For instance, if the name of your application is example-app:
cat .godir
# example-app
  • or, it should use godep to manage dependencies, and you must include the generated Godep directory in your git repository.

See the Go buildpack documentation for more details.

Example applications

  • Gogs (Go Git Service), the painless self-hosted Git Service written in Go.
  • example-go-app, a simple Go application to get you started.
